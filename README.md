
# Optimizing IT Support Team Performance Using Analytics

## Project Overview
IT support teams handle hundreds of tickets daily, but delays in responses, long resolution times, and inefficient workflows can reduce productivity and customer satisfaction.

This project uses **data analytics and visualization** to analyze IT support ticket data, identify performance bottlenecks, and provide actionable insights to improve service efficiency. The goal is to help teams resolve issues faster, prioritize critical requests, and deliver better customer experiences.



## Problem Statement
The main objective of this project is to:

- Analyze IT support ticket trends
- Reduce resolution time
- Improve first response time
- Identify frequently occurring issues
- Measure customer satisfaction
- Optimize team workflow and resource allocation

By leveraging data, we aim to make smarter operational decisions rather than relying on assumptions.



## Dataset Information

**Dataset:** Customer Support Ticket Dataset  
**Source:** Kaggle  
**Size:** 8000+ rows and 17 columns  

The dataset contains customer support requests related to:

- Hardware issues  
- Software bugs  
- Network problems  
- Account access issues  
- Data loss  
- Product and billing inquiries  

Each record captures ticket details, timing metrics, and customer feedback.
